# Projeto

A documentacao principal do repositorio esta no [README da raiz](../README.md).

## Comandos rapidos

Os comandos abaixo partem da raiz do repositorio.

Frontend:

```bash
cd projeto
npm run dev
```

Backend:

```bash
cd projeto/backend
npm run dev
```

Seed do super usuario:

```bash
cd projeto/backend
npm run seed:super
```
